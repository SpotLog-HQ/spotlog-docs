# My Spots - Edit A Spot

It is possible to edit a Spot from the my spots screen, simply click the edit icon against the spot you would like to edit. 

<a href="/images/my-spots/my-spots-edit.png" target="_blank">
  <img
    src="/images/my-spots/my-spots-edit.png"
    class="doc-img md"
    alt="SpotLog – Calendar screen"
  />
</a>

You will then be taken to the edit screen, where you can edit the details of the spot. This can be done as many times as you like however the history of your changes are not kept.
