This section allows you to record details about the locomotive itself

This could include 

- The locomotive number or headcode
- The locomotive name
- The class of the locomotive
- The operator 
- The locomotive type
- The condition of the locomotive
- Was the locomotive tender first (only visible when locomotive type = Steam)
